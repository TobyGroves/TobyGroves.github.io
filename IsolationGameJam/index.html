I set up the isolation game jam to kill the boredom of being stuck inside. I publicised it on a bunch of discord servers and on twitter.

My entry to the jam was a game where you use a hoover to clean up a western town. I was using this as an excuse to create a awsome looking sand shader. After alot of googleing and alot of tweaking I managed to get something that looked similar to journey. I was then faced with the issue of "How the hell do I put this in a game about cleaning ?". As you can see the obvious answer was you clean up a western themed town that has a sandy floor... Duhh!

While the game isn't particularly fun and dosent make any sense, I was very proud of the sand shader. You can see the source code here on this gist: click here
