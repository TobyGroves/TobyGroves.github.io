About You Complete Me

You Complete me is a game for completionists there are around 75 procedurally generate minigame / challenges of 4 types spread across a archipelligo of islands which take around 5 hours to complete. In order to travel between the minigames / challenges you fly a drone with a winch attached.

Developing this game has required me to learn new skills and solve problems that I have never come across before AWS being the main culprit as I had never done any network or amazon stuff before.

If youd like to follow the game's development head over to the Red Trombone Games Website for some infrequent blog posts about what I'm doing.
