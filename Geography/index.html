Squiggly Pi is a meta language that transcompiles into multiple major object oriented languages.

Version 1.1 transcompiles to Java, C++, Python, C#, Scala, Groovy, JavaScript/ECMAScript, CoffeeScript, PHP, Perl, Ruby and Swift.

Squiggly Pi has 2 modes: Standard and Hardcore. Hardcore mode only allows you to have single letter variable names and no english words are used in the syntax.

Here is Hello World in Squiggly pi.

$< -" HelloWorld"\
